### Tech Stack - React + TypeScript + Vite

#### To run the project:

1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
4. Open `localhost:5173` in your browser
5. Enjoy!
6. Run `npm run build` to build the project
